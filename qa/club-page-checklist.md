# QA Checklist For Club Pages

Bruk denne sjekklisten før en klubbside regnes som ferdig.

## Språk og tone
- [ ] Alt innhold er skrevet i konsekvent Bokmål.
- [ ] Teksten høres naturlig ut for en norsk golfklubb.
- [ ] Tonen er lokal, troverdig og uten markedsføringsfloskler.
- [ ] Samme poeng er ikke gjentatt unødvendig i flere seksjoner.

## Fakta og kildebruk
- [ ] Alle fakta er verifisert mot reelle kilder.
- [ ] Ingen fakta virker oppdiktet, antatt eller AI-generert.
- [ ] Historikk, baneinfo, fasiliteter og kontaktinformasjon stemmer.
- [ ] Datoer og årstall er kontrollert.

## Reviews og sosial bevisføring
- [ ] Bare ekte anmeldelser er brukt.
- [ ] Ingen falske reviews eller omskrevne “kundeuttalelser” er lagt inn.
- [ ] Review-navn, rating og tekst er tro mot kilden.
- [ ] Review-seksjonen er skjult hvis materialet er for svakt.

## Nyheter og aktivitet
- [ ] Nyhetsseksjonen inneholder bare reelle nyheter eller faktiske klubboppdateringer.
- [ ] Nyhetsinnholdet er fortsatt aktuelt nok til å vises.
- [ ] Ingen generiske placeholder-kort er igjen.
- [ ] Datoer, kategorier og omtale stemmer med kildematerialet.

## Seksjonskvalitet
- [ ] Ingen seksjoner føles tomme eller kunstig utfylte.
- [ ] Ingen dupliserte seksjoner gjentar samme funksjon.
- [ ] Svake seksjoner er skjult eller nedtonet i stedet for å fylles med fluff.
- [ ] Hver seksjon har en tydelig rolle på siden.

## Praktisk informasjon
- [ ] Priser er korrekte.
- [ ] Greenfee, medlemskap og spillerett er korrekt oppgitt.
- [ ] Bookinginformasjon er korrekt.
- [ ] Betalingsinformasjon er korrekt.
- [ ] Fasiliteter er korrekt oppført.
- [ ] Kontaktinformasjon er korrekt oppført.

## CTA-er og lenker
- [ ] CTA-tekster er realistiske og konkrete.
- [ ] Ingen generiske eller misvisende CTA-er er brukt.
- [ ] Synlig lenketekst passer konteksten.
- [ ] Lenker peker til riktig sted.

## Bilder og visuell relevans
- [ ] Bildene tilhører riktig klubb.
- [ ] Bildetekster / alt-tekst er saklige og presise.
- [ ] Ingen feil klubbbilder eller irrelevante illustrasjoner er brukt.

## Sluttkontroll
- [ ] Datafilen stemmer med briefen.
- [ ] Siden er manuelt forhåndsvist.
- [ ] `npm run build` består.
